# Pokemon Configuration

The Pokemon mod uses a YAML configuration file to control various aspects of Pokemon behavior, spawn rates, and game mechanics.

## Configuration File Location

The main configuration file is located at:

```
plugins/PokemonMod/config.yml
```

This file is automatically generated when the mod is first loaded.

## Configuration Sections

### Pokemon Settings

Controls general Pokemon behavior:

```yaml
pokemon:
  max-level: 100
  min-level: 1
  default-spawn-level: 5
  level-range: 10
  exp-multiplier: 1.0
  shiny-rate: 0.001
```

#### Options

**`max-level`**
- **Type:** `int`
- **Default:** `100`
- **Range:** `1-1000`
- **Description:** Maximum level a Pokemon can reach

**`min-level`**
- **Type:** `int`
- **Default:** `1`
- **Range:** `1-100`
- **Description:** Minimum level for wild Pokemon spawns

**`default-spawn-level`**
- **Type:** `int`
- **Default:** `5`
- **Range:** `1-100`
- **Description:** Default level for spawned Pokemon if no level range is specified

**`level-range`**
- **Type:** `int`
- **Default:** `10`
- **Range:** `1-50`
- **Description:** Level variation for wild Pokemon (spawn level Â± this value)

**`exp-multiplier`**
- **Type:** `double`
- **Default:** `1.0`
- **Range:** `0.1-10.0`
- **Description:** Multiplier for experience gained (1.0 = normal, 2.0 = double)

**`shiny-rate`**
- **Type:** `double`
- **Default:** `0.001` (1 in 1000)
- **Range:** `0.0-1.0`
- **Description:** Probability of a Pokemon being shiny

### Spawn Settings

Controls how Pokemon spawn in the world:

```yaml
spawn:
  enabled: true
  spawn-rate: 0.1
  max-per-chunk: 5
  despawn-distance: 64
  spawn-interval: 300
  respect-biomes: true
```

#### Options

**`enabled`**
- **Type:** `boolean`
- **Default:** `true`
- **Description:** Enable/disable natural Pokemon spawning

**`spawn-rate`**
- **Type:** `double`
- **Default:** `0.1`
- **Range:** `0.0-1.0`
- **Description:** Global spawn rate multiplier (0.0 = no spawns, 1.0 = maximum)

**`max-per-chunk`**
- **Type:** `int`
- **Default:** `5`
- **Range:** `1-20`
- **Description:** Maximum Pokemon that can spawn in a single chunk

**`despawn-distance`**
- **Type:** `int`
- **Default:** `64`
- **Range:** `16-128`
- **Description:** Distance in blocks before Pokemon despawn (if no players nearby)

**`spawn-interval`**
- **Type:** `int`
- **Default:** `300`
- **Range:** `60-3600`
- **Description:** Time in seconds between spawn attempts

**`respect-biomes`**
- **Type:** `boolean`
- **Default:** `true`
- **Description:** Whether Pokemon spawns should respect biome types

### Battle Settings

Controls battle mechanics:

```yaml
battle:
  turn-timeout: 30
  allow-items: true
  allow-switching: true
  critical-hit-rate: 0.0625
  same-type-bonus: 1.5
```

#### Options

**`turn-timeout`**
- **Type:** `int`
- **Default:** `30`
- **Range:** `10-120`
- **Description:** Time in seconds before a turn times out (player forfeits)

**`allow-items`**
- **Type:** `boolean`
- **Default:** `true`
- **Description:** Whether items can be used during battles

**`allow-switching`**
- **Type:** `boolean`
- **Default:** `true`
- **Description:** Whether Pokemon can be switched during battle

**`critical-hit-rate`**
- **Type:** `double`
- **Default:** `0.0625` (6.25%)
- **Range:** `0.0-1.0`
- **Description:** Base critical hit chance

**`same-type-bonus`**
- **Type:** `double`
- **Default:** `1.5`
- **Range:** `1.0-3.0`
- **Description:** Damage multiplier when move type matches Pokemon type (STAB)

### Economy Settings

Controls Pokemon-related economy features:

```yaml
economy:
  pokeball-cost: 200
  potion-cost: 100
  sell-pokemon: true
  sell-multiplier: 0.5
```

#### Options

**`pokeball-cost`**
- **Type:** `int`
- **Default:** `200`
- **Range:** `0-10000`
- **Description:** Cost to buy a standard Poke Ball

**`potion-cost`**
- **Type:** `int`
- **Default:** `100`
- **Range:** `0-10000`
- **Description:** Cost to buy a Potion

**`sell-pokemon`**
- **Type:** `boolean`
- **Default:** `true`
- **Description:** Whether players can sell Pokemon to NPCs

**`sell-multiplier`**
- **Type:** `double`
- **Default:** `0.5`
- **Range:** `0.0-1.0`
- **Description:** Multiplier for Pokemon sell price (based on level/stats)

## Example Configuration

Here's a complete example configuration:

```yaml
pokemon:
  max-level: 100
  min-level: 1
  default-spawn-level: 5
  level-range: 10
  exp-multiplier: 1.5
  shiny-rate: 0.002

spawn:
  enabled: true
  spawn-rate: 0.15
  max-per-chunk: 8
  despawn-distance: 80
  spawn-interval: 240
  respect-biomes: true

battle:
  turn-timeout: 45
  allow-items: true
  allow-switching: true
  critical-hit-rate: 0.0625
  same-type-bonus: 1.5

economy:
  pokeball-cost: 150
  potion-cost: 80
  sell-pokemon: true
  sell-multiplier: 0.6
```

## Reloading Configuration

You can reload the configuration without restarting the server:

```
/pokemon reload
```

This command reloads:
- `config.yml` - Main configuration
- Pokemon species definitions
- Move definitions

**Note:** Some changes (like spawn settings) may take effect on the next spawn cycle.

## Per-World Configuration

You can create world-specific configurations by creating:

```
plugins/PokemonMod/worlds/<world-name>/config.yml
```

World-specific configs override the main config for that world only.
